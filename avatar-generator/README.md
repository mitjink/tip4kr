# Генератор случайных аватаров

## Описание приложения

Приложение "Генератор случайных аватаров" создает уникальных пиксельных персонажей с помощью алгоритма случайной комбинации характеристик. При нажатии кнопки "Сгенерировать случайный аватар" система случайным образом выбирает один вариант из каждой категории: цвет кожи (4 варианта), цвет глаз (3 варианта), стиль прически (12 вариантов) и цвет одежды (4 варианта). Все аватары были нарисованы в программе **Pixel Studio**. Все изображения накладываются слоями в определенном порядке для создания целостного персонажа.
